# 🌌 n3bu1a Suite: gxtop & n3matrix

> A celestial system monitor and star-rain visualizer for Arch Linux and beyond.

The **n3bu1a Suite** transforms your terminal into a deep-space command center. Built for the **11th Gen i5-1135G7** and optimized for a "Starred Galaxy" aesthetic, these tools bring high-performance system tracking and 3D parallax visuals to your workspace.

---

## 🛠 Included Tools

### ☄️ n3matrix (Nebula Rain)
A modern evolution of the classic matrix rain. 
- **3D Parallax:** Stars fall at variable speeds for depth.
- **Nebula Trails:** Faint dust particles follow the falling stars.
- **Shimmering Palette:** Deep purples (#af00ff) and cyans (#00d7ff).

### 🖥️ gxtop (GalaxyTop)
A high-detail system monitor inspired by `btop`.
- **Core Tracking:** Real-time monitoring of all CPU threads.
- **Galaxy View:** A live-rendering starfield in the corner.
- **Nebula Rotor:** A 3D-spinning ASCII "NEBULA" logo that reacts to system state.
